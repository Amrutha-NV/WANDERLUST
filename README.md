A full-stack Airbnb-style web platform where users can create, explore, and manage property listings with secure authentication, image hosting, reviews, and interactive map integration.

1.Developed a full-stack Airbnb-style web platform using Node.js, Express.js, MongoDB, and EJS.

2.Implemented secure user authentication and authorization using Passport.js with session management.

3.Designed **CRUD functionality** for creating, updating, and deleting property listings with ownership validation.

4.Integrated **Cloudinary** for reliable image upload, storage, and retrieval.

5. Added **interactive location maps** using **Leaflet.js** for dynamic property visualization.
   
6. Built a **review and rating system** enabling users to provide feedback on listings.
   
7. Ensured smooth user experience with **error handling**, **flash messaging**, and a **responsive UI** using **Bootstrap 5**.
   
8.Deployed the complete application on **Render** with environment-based configuration and MongoDB Atlas integration.
